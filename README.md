# PASS

PASS is an umbrella repository for a collection of public-health sequencing and metadata-curation projects. Each sub-directory is an independent, PASS-affiliated project with its own documentation, dependencies, and usage instructions. This top-level README serves as an entry point that introduces the projects and provides citation information for the papers associated with them.

## Projects

| Project | Description |
|---|---|
| [SGMC](./SGMC/) | Search-based Geographic Metadata Curation pipeline that refines sequencing-institution information in NCBI SRA metadata to support public health. |
| [DPS](./DPS/) | Placeholder — details coming soon. |

See each project's own README for installation and usage details.

## How to cite

If you use a PASS project in your work, please cite the corresponding paper.

### SGMC

> Zhao, K., Farrell, K., Mashiku, M., Abay, D., Tang, K., Oberste, M. S., & Burns, C. C. (2023). A search-based geographic metadata curation pipeline to refine sequencing institution information and support public health. *Frontiers in Public Health*, 11, 1254976. https://doi.org/10.3389/fpubh.2023.1254976

```bibtex
@article{zhao2023sgmc,
  author  = {Zhao, Kun and Farrell, Kelly and Mashiku, Melchy and Abay, Dawit and Tang, Kun and Oberste, M. Steven and Burns, Cara C.},
  title   = {A search-based geographic metadata curation pipeline to refine sequencing institution information and support public health},
  journal = {Frontiers in Public Health},
  volume  = {11},
  pages   = {1254976},
  year    = {2023},
  doi     = {10.3389/fpubh.2023.1254976},
  url     = {https://doi.org/10.3389/fpubh.2023.1254976}
}
```
